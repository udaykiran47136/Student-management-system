# Student-management-system

The Student Management System is a desktop-based application developed using Python (Tkinter) and SQLite to efficiently manage student records in educational institutions. The system provides a user-friendly graphical interface that allows administrators to perform complete CRUD operations (Create, Read, Update, Delete) on student data.

It includes features such as student enrollment management, class and batch tracking, mobile number storage, and monthly fee payment monitoring from January to December. The project demonstrates strong understanding of GUI development, database connectivity, SQL queries, and event-driven programming in Python.

This project was built to gain practical experience in integrating frontend interfaces with backend databases while ensuring data validation and structured record management.
